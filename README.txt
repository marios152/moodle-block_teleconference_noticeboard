moodle-block_teleconference_noticeboard
---------------------------------------

Description:
------------
A block that will create label announcements for the users of the course 
with details such as date, time, teleconference number and teleconference password 
along with the link of the teleconference.

Installation:
-------------
Extract and upload the 'moodle-block_teleconference_noticeboard' folder into moodle/blocks/

Supported Moodle versions:
--------------------------
I have tested the plugin on a clean install of Moodle 3.0
